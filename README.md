# Terraform Infrastructure 🏗️

AWS cloud infrastructure provisioning using Terraform
Infrastructure as Code — Phase 4 of the Roboshop project.

## What's Included
- EC2 instance provisioning
- VPC, subnets, security groups
- Route53 DNS configuration
- Packer AMI creation
- Terraform modules and variables
